# CSE 151B SP26 Competition — Team Submission

## Hardware

| Item | Detail |
|------|--------|
| GPU | NVIDIA A100 (40 GB) via Google Colab |
| Approximate total inference time | ~3–4 hours for the full private test set (5 samples × USC voting) |
| Approximate SFT training time | ~2–3 hours (3 epochs on ~1,000 examples) |
| Approximate GRPO training time | ~4–6 hours (2 epochs on 3,000 problems) |

## Pipeline Overview

1. **Data collection** (`scripts/get_data.py`) — filter correct DeepSeek-V4-Pro traces from `deepseek-v4-pro_all_1.jsonl` using the official judger
2. **SFT** (`scripts/fine_tune.py`) — QLoRA fine-tune `Qwen/Qwen3-4B-Thinking-2507` on the filtered traces (LoRA r=16)
3. **RL data prep** (`scripts/prepare_rl_data.py`) — build a 3,000-problem GRPO training set from NuminaMath + MATH + MMLU
4. **GRPO** (`scripts/grpo.py`) — GRPO training on top of the merged SFT model (LoRA r=32)
5. **Inference** (`run_inference.py`) — vLLM + LoRA, 5 candidates per question, USC voting

## Model Weights

Both LoRA adapters are hosted on HuggingFace Hub.

### Download / Setup

The adapters are loaded automatically by `run_inference.py` via the `hf_adapter` argument. No manual download needed — vLLM pulls them from the Hub.

If you want to inspect them locally:

```bash
pip install huggingface_hub
huggingface-cli download YOUR_HF_USERNAME/qwen3-4b-grpo-lora --local-dir ./adapters/grpo
huggingface-cli download YOUR_HF_USERNAME/qwen3-4b-sft-lora  --local-dir ./adapters/sft
```

Place the adapter files anywhere; pass the local path (or Hub ID) to `run_inference()`.

### Adapter details

| Adapter | Base model | LoRA r | Stage |
|---------|-----------|--------|-------|
| `YOUR_HF_USERNAME/qwen3-4b-sft-lora` | Qwen/Qwen3-4B-Thinking-2507 | 16 | After SFT |
| `YOUR_HF_USERNAME/qwen3-4b-grpo-lora` | Qwen/Qwen3-4B-Thinking-2507 (+ merged SFT) | 32 | After GRPO |

The **GRPO adapter** (`adapter_model (1).safetensors`) is used for the final submission.

## Reproducing Inference

### Install dependencies

```bash
pip install vllm transformers peft sympy scipy \
            "antlr4-python3-runtime==4.11.1" tqdm
```

### Run

```python
from run_inference import run_inference

run_inference(
    data_path  = "data/private.jsonl",
    output_dir = "results/",
    hf_adapter = "YOUR_HF_USERNAME/qwen3-4b-grpo-lora",
)
```

Or from the command line:

```bash
python run_inference.py \
    --data    data/private.jsonl \
    --out     results/ \
    --adapter YOUR_HF_USERNAME/qwen3-4b-grpo-lora \
    --voting  usc
```

The function writes three CSVs (`submission_majority.csv`, `submission_weighted.csv`, `submission_usc.csv`) and returns the path to the chosen one. **Our final submission uses USC voting.**

### Key hyperparameters (final values)

## Uploading Weights (team reference)

See the instructions below if you need to re-upload the adapters.

```python
from huggingface_hub import login, HfApi
import shutil, os

login()   # paste your HF token

# ── Upload GRPO adapter ─────────────────────────────────────────────────────
# 1. Put adapter_model.safetensors (GRPO) and adapter_config.json in one folder
os.makedirs("upload_grpo", exist_ok=True)
shutil.copy("adapter_model (1).safetensors", "upload_grpo/adapter_model.safetensors")
# adapter_config.json is generated by the training script — copy it too
shutil.copy("checkpoints/grpo_checkpoint/final_adapter/adapter_config.json",
            "upload_grpo/adapter_config.json")

api = HfApi()
api.create_repo("YOUR_HF_USERNAME/qwen3-4b-grpo-lora", exist_ok=True)
api.upload_folder(
    folder_path="upload_grpo",
    repo_id="YOUR_HF_USERNAME/qwen3-4b-grpo-lora",
    repo_type="model",
)

# ── Upload SFT adapter ──────────────────────────────────────────────────────
os.makedirs("upload_sft", exist_ok=True)
shutil.copy("adapter_model.safetensors", "upload_sft/adapter_model.safetensors")
shutil.copy("deepseekv4-pro/qwen3-4b-sft-lora/checkpoint-396/adapter_config.json",
            "upload_sft/adapter_config.json")

api.create_repo("YOUR_HF_USERNAME/qwen3-4b-sft-lora", exist_ok=True)
api.upload_folder(
    folder_path="upload_sft",
    repo_id="YOUR_HF_USERNAME/qwen3-4b-sft-lora",
    repo_type="model",
)
```
